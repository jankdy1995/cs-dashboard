# NeoTaste CS KPI Dashboard

Automatisch generiertes Customer-Success-Dashboard. Die Daten stammen aus der
KPI-Excel in Google Drive; `build_dashboard.py` erzeugt daraus `index.html`
(öffentliche Variante ohne vertrauliche Kostendaten, Flag `--public`).

Live: https://jankdy1995.github.io/cs-dashboard/
Update: täglich ~7:00 Uhr (Europe/Berlin) via Claude Scheduled Task.
